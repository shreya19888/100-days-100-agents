# 🌤️ Climate Pulse AI

**Day 002 of the 100 Days • 100 AI Agents Challenge**

🚀 **Live Demo:** https://climate-pulse-ai-three.vercel.app/

**GitHub Repository:** https://github.com/shreya19888/100-days-100-agents

Climate Pulse AI is an AI-powered outdoor safety assistant that helps workers make safer decisions using live weather conditions, a deterministic climate risk engine, and GPT-5.5 to generate personalized occupational safety recommendations.

---

## 🌍 Live Demo

Try the application here:

**https://climate-pulse-ai-three.vercel.app/**

### Demo Flow

1. Select your occupation
2. Search for any city
3. View live weather conditions
4. Receive an AI-generated safety assessment
5. Get personalized hydration, PPE, and work schedule recommendations
