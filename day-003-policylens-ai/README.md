## 🌍 Day 003 — PolicyLens AI

**A Multilingual Agentic RAG Platform for Global HR Policies**

PolicyLens AI helps employees instantly find answers from global HR policies, benefits, and compliance documents using AI. Employees can ask questions in their preferred language, while the system retrieves the most relevant policy sections using semantic search and returns grounded answers with citations.

### ✨ Features

- 🌍 Multilingual question answering
- 🤖 Agentic Retrieval-Augmented Generation (RAG)
- 📄 Semantic search across HR policy documents
- 🔍 Source citations for every answer
- 📚 Enterprise-ready document intelligence

**Tech Stack**

- Next.js
- TypeScript
- OpenAI GPT-5.5
- OpenAI Embeddings
- LangChain
- LangGraph
- Tailwind CSS
